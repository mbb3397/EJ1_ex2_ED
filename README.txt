Dentro de las clases hemos generado los comentarios explicando
las funcionalidades de los distintos métodos, los parámetros que requieren
y la estructura de clases que tiene el proyecto.
El proyecto consta de clases para películas
Clases para los usuarios 
y métodos de Añadir, Borrar y Comprar películas que serán para un carrito de la compra
por otro lado hay clases que interactúan directamente con la base de datos
para obtener información, insertar y eliminar de la base de datos las películas y los usuarios.
